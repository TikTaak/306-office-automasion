## set env var for github token:
### Power shell:
- add:
  
    ```setx GITHUB_TOKEN "your_personal_token_here"```

- remove:

    ```setx GITHUB_TOKEN ""```

### Cmd:


- add:
  
    ```set GITHUB_TOKEN=your_personal_access_token```

- remove:
  
    ```set GITHUB_TOKEN=```
